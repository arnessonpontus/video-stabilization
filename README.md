# Real-Time Video Stabilization

> Course: TNM089

## Dependencies

Make sure opencv and numpy and matplotlib is installed

```
pip install opencv-python numpy matplotlib
```

## How to use

To use with webcam:

```
python main.py
```

> if client crashes, change VideoCapture number on line 22 in main.py to 1

To use from video path:

```
python main.py [PATH TO VIDEO]
```
